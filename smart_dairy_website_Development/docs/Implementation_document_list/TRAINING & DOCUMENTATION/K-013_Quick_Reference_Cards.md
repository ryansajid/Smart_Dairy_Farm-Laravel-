## 1. Introduction

### 1.1 Purpose

Quick Reference Cards (QRCs) are compact, job-specific guides designed to provide Smart Dairy Ltd. employees with immediate access to essential procedures, shortcuts, and critical information. These cards serve as portable reminders that reduce training time, minimize errors, and boost confidence among system users.

### 1.2 Document Objectives

- Standardize the format and content of quick reference materials
- Provide role-specific guidance for all Smart Dairy system users
- Establish print and digital distribution standards
- Define maintenance and update procedures

### 1.3 Target Audience

| Audience | Purpose |
|----------|---------|
| Farm Workers | Daily operational tasks (Bilingual: Bengali/English) |
| Farm Supervisors | Dashboard management and approvals |
| Sales Team | Order processing and customer management |
| Warehouse Staff | Inventory handling and safety |
| Accounting Staff | Financial procedures and reporting |
| System Administrators | System maintenance and troubleshooting |

### 1.4 Format and Distribution

**Physical Cards:**
- Laminated, pocket-sized cards for field use
- Posted at workstations for quick access
- Distributed during onboarding and training

**Digital Versions:**
- PDF files for download and printing
- Mobile-optimized versions for smartphone access
- WhatsApp shareable images for field staff

### 2.5 Icons and Visuals

**Icon Set:** Phosphor Icons or Heroicons (consistent line style)

| Category | Icons | Purpose |
|----------|-------|---------|
| Actions | Login, Search, Save, Print | Step indicators |
| Navigation | Home, Back, Menu, User | Navigation aids |
| Status | Check, Warning, Info, Error | Status indicators |
| Communication | Phone, Message, Email | Contact methods |
| Objects | Cow, Milk, Box, Calculator | Role-specific items |

**Visual Guidelines:**
- Use 24px icons for main steps
- Use 16px icons for tips and notes
- Maintain consistent stroke width (2px)
- Use brand colors for icons
