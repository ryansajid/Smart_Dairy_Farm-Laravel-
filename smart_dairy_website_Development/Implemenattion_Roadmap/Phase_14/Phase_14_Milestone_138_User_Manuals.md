### 4.2 Admin User Manual - Sample Chapter

```markdown
# Chapter 3: User Management

## 3.1 Overview

As a System Administrator, you have full control over user accounts in the
Smart Dairy Digital Portal. This chapter covers:

- Creating new user accounts
- Assigning roles and permissions
- Managing existing users
- Deactivating accounts

## 3.2 Understanding Roles

The system includes the following user roles:

| Role | Description | Access Level |
|------|-------------|--------------|
| Administrator | Full system access | All modules |
| Farm Manager | Farm operations management | Farm, Animals, Collections |
| Farm Worker | Day-to-day operations | Collections, Animals (view) |
| Customer | Consumer access | Orders, Products, Account |
| B2B Partner | Business customer | B2B Portal, Orders, Reports |
| Driver | Delivery operations | Deliveries, Navigation |

## 3.3 Creating a New User

### Step-by-Step Guide

1. **Navigate to User Management**
   - From the main dashboard, click **Settings** in the navigation menu
   - Select **Users** from the dropdown

   ![User Menu Navigation](./images/admin/user-menu.png)

2. **Click "Create New User"**
   - Click the **+ Create User** button in the top right corner

   ![Create User Button](./images/admin/create-user-btn.png)

3. **Enter User Details**

   Fill in the required information:

   | Field | Description | Required |
   |-------|-------------|----------|
   | Full Name | User's complete name | Yes |
   | Email | Login email address | Yes |
   | Phone | Contact number | Yes |
   | Role | Select from dropdown | Yes |
   | Farm | Assign to farm (if applicable) | Conditional |

   ![New User Form](./images/admin/new-user-form.png)

4. **Set Initial Password**

   Choose one of the following options:
   - **Generate Random Password**: System creates a secure password
   - **Set Custom Password**: Enter a password manually

   > **Note**: Users will be prompted to change their password on first login.

5. **Configure Permissions** (Optional)

   For granular access control:
   - Click **Advanced Permissions**
   - Check/uncheck specific module access
   - Click **Save Permissions**

6. **Send Welcome Email**

   - Check **Send Welcome Email** to notify the user
   - The email includes:
     - Login URL
     - Username (email)
     - Temporary password
     - Getting started guide

7. **Click "Create User"**

   - Review all information
   - Click **Create User** to finalize

   ![Create User Success](./images/admin/user-created.png)

### Best Practices

✅ **Do:**
- Use official company email addresses
- Assign the minimum necessary permissions
- Regularly audit user accounts
- Document role assignments

❌ **Don't:**
- Share login credentials
- Create generic accounts
- Leave inactive accounts active
- Skip the welcome email

## 3.4 Managing Existing Users

### Viewing User List

1. Navigate to **Settings > Users**
2. Use filters to find specific users:
   - Role filter
   - Status filter (Active/Inactive)
   - Farm filter
   - Search by name or email

### Editing User Information

1. Click on the user's name to open their profile
2. Click **Edit** in the top right
3. Modify the necessary fields
4. Click **Save Changes**

### Changing User Role

> **Warning**: Changing a user's role immediately affects their access.

1. Open user profile
2. Click **Change Role**
3. Select new role from dropdown
4. Confirm the change
5. User will be notified of the role change

### Resetting Password

1. Open user profile
2. Click **Reset Password**
3. Choose method:
   - Send reset link via email
   - Generate new temporary password
4. Confirm the action

## 3.5 Deactivating Users

When an employee leaves or access should be revoked:

1. Open user profile
2. Click **Deactivate Account**
3. Select reason:
   - Employee departure
   - Security concern
   - Account consolidation
   - Other
4. Choose data handling:
   - Preserve all data
   - Transfer data to another user
   - Archive and remove
5. Confirm deactivation

> **Note**: Deactivated accounts can be reactivated within 90 days.

## 3.6 Bulk User Operations

For managing multiple users at once:

### Bulk Import

1. Go to **Settings > Users > Import**
2. Download the CSV template
3. Fill in user details
4. Upload completed CSV
5. Review import preview
6. Confirm import

**CSV Format:**
```csv
name,email,phone,role,farm_id
"John Smith",john@farm.com,+8801234567890,farm_manager,1
"Jane Doe",jane@farm.com,+8801234567891,farm_worker,1
```

### Bulk Export

1. Select users from the list
2. Click **Export Selected**
3. Choose format (CSV, Excel, PDF)
4. Download the file

## 3.7 Troubleshooting

### Common Issues

| Issue | Cause | Solution |
|-------|-------|----------|
| User can't login | Wrong password | Reset password |
| User can't see module | Missing permission | Check role permissions |
| Duplicate email error | Email already exists | Use different email |
| Welcome email not received | Email blocked | Check spam folder |

### Error Messages

**"Email already registered"**
- Another user has this email
- Solution: Use a different email or merge accounts

**"Invalid role assignment"**
- Role requires specific prerequisites
- Solution: Ensure user meets role requirements

---

   - প্রধান ড্যাশবোর্ড থেকে, নেভিগেশন মেনুতে **সেটিংস** ক্লিক করুন
   - ড্রপডাউন থেকে **ব্যবহারকারী** নির্বাচন করুন

   ![ব্যবহারকারী মেনু নেভিগেশন](./images/admin/user-menu-bn.png)

২. **"নতুন ব্যবহারকারী তৈরি করুন" ক্লিক করুন**
   - উপরের ডান কোণে **+ ব্যবহারকারী তৈরি করুন** বাটন ক্লিক করুন

৩. **ব্যবহারকারীর তথ্য লিখুন**

   প্রয়োজনীয় তথ্য পূরণ করুন:

   | ক্ষেত্র | বিবরণ | আবশ্যক |
   |--------|-------|--------|
   | পূর্ণ নাম | ব্যবহারকারীর সম্পূর্ণ নাম | হ্যাঁ |
   | ইমেইল | লগইন ইমেইল ঠিকানা | হ্যাঁ |
   | ফোন | যোগাযোগ নম্বর | হ্যাঁ |
   | ভূমিকা | ড্রপডাউন থেকে নির্বাচন করুন | হ্যাঁ |
   | ফার্ম | ফার্মে বরাদ্দ করুন (প্রযোজ্য হলে) | শর্তসাপেক্ষ |

৪. **প্রাথমিক পাসওয়ার্ড সেট করুন**

   নিম্নলিখিত বিকল্পগুলির মধ্যে একটি বেছে নিন:
   - **র্যান্ডম পাসওয়ার্ড তৈরি করুন**: সিস্টেম একটি নিরাপদ পাসওয়ার্ড তৈরি করে
   - **কাস্টম পাসওয়ার্ড সেট করুন**: ম্যানুয়ালি একটি পাসওয়ার্ড লিখুন

   > **দ্রষ্টব্য**: প্রথম লগইনে ব্যবহারকারীদের পাসওয়ার্ড পরিবর্তন করতে বলা হবে।

৫. **"ব্যবহারকারী তৈরি করুন" ক্লিক করুন**

   - সমস্ত তথ্য পর্যালোচনা করুন
   - চূড়ান্ত করতে **ব্যবহারকারী তৈরি করুন** ক্লিক করুন

### সর্বোত্তম অনুশীলন
